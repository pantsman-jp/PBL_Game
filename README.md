# PBL_Game

## Installation
```PowerShell
> git clone https://github.com/pantsman-jp/PBL_Game
```

## Usage
Require `Python3`, `pyxel`.

Install `pyxel` via
```PowerShell
> pip install pyxel
```

To run
```PowerShell
> cd src
> python src/main.py
```

## 開発者へ
### 注意
**コミット時はディレクトリ位置に注意**

### Pyxel Editor の使い方
**/src** ディレクトリまで移動してから実行すること。
```PowerShell
> pyxel edit assets.pyxres
```
