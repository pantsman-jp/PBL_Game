# PBL_Game

## インストール
```PowerShell
> git clone https://github.com/pantsman-jp/PBL_Game
```

## 使い方
`Python3`, `Pygame` が必要。

`Pygame` は以下でインストール。
```PowerShell
> pip install pygame
```

実行方法は以下。
```PowerShell
> python src/main.py
```
