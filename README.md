# PBL_Game

## インストール
```PowerShell
> git clone https://github.com/pantsman-jp/PBL_Game
```

## 使い方
`Python3`, `pyxel` が必要。

`pyxel` は以下でインストール。
```PowerShell
> pip install pyxel
```

実行方法は以下。
```PowerShell
> python src/main.py
```

## 操作方法
`z` : 会話開始、選択  
`q` : 会話から離脱  
`s` : セーブ  

## 開発者へ
### Pyxel Editor の使い方
**/src** ディレクトリまで移動してから実行すること。
```PowerShell
> pyxel edit assets.pyxres
```
