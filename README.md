# PBL_Game

## Installation
```PowerShell
> git clone https://github.com/pantsman-jp/PBL_Game
```

## Usage
Require `Python3`, `pyxel`.

Install via
```PowerShell
> pip install pyxel
```

To run
```PowerShell
> python src/main.py
```

To quit, press `q` key.
