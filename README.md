# PBL_Game

## 概要
このゲームは、**宇宙工学・宇宙理学の面白さを高校生に紹介する** というコンセプトで開発されています。

## インストール方法
```PowerShell
git clone https://github.com/pantsman-jp/PBL_Game
```

## 使い方
`Python3.13.9`, `Pygame2.6.1` が必要。

`Pygame` は以下でインストール。
```PowerShell
pip install pygame
```

実行方法は以下。
```PowerShell
python main.py
```
