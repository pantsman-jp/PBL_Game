# PBL_Game

## インストール
```PowerShell
> git clone https://github.com/pantsman-jp/PBL_Game
```

## 使い方
`Python3`, `Pyxel`, `Pillow` が必要。

`pyxel` は以下でインストール。
```PowerShell
> pip install pyxel
> pip install Pillow
```

実行方法は以下。
```PowerShell
> python src/main.py
```

## 開発者へ
### 注意
**コミット時はディレクトリ位置に注意**

### Pyxel Editor の使い方
**/src** ディレクトリまで移動してから実行すること。
```PowerShell
> pyxel edit assets.pyxres
```
