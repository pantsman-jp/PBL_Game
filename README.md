# PBL_Game

## Installation
```
git clone https://github.com/pantsman-jp/PBL_Game
```

## Usage
Require `Python3`, `pyxel`.

Install `pyxel` via
```
pip install pyxel
```

Open Pyxel Editor
```PowerShell
> pyxel edit Pyxelリソースファイル
公式URL
https://github.com/kitao/pyxel/blob/main/docs/README.ja.md
```

To run
```
python src/main.py
```
