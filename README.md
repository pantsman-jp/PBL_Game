# PBL_Game

## Installation
```PowerShell
> git clone https://github.com/pantsman-jp/PBL_Game
```

## Usage
Require `Python3`, `pyxel`.

Install `pyxel` via
```PowerShell
> pip install pyxel
```

To run
```PowerShell
> cd src
> python src/main.py
```

> [!CAUTION]
> コミット時は以下を実行して親ディレクトリまで移動すること
> `cd ..`
